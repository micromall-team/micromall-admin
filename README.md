# micromall-admin

# micromall商城后台管理系统

```bash
#克隆项目
git clone https://github.com/micromall-team/micromall-admin.git

#安装依赖
npm install

#运行项目
npm run dev

#打包项目
npm run build

#项目说明
本项目采用Vue3 + vite + TypeScript + Vue-router4 +Vuex4 为核心，借助开源UI框架element plus的风格搭建的一个商城后台管理平台。

#Vue3.0 官网
https://vue3js.cn/docs/

#Element Plus 官网
https://element-plus.org/#/zh-CN/guide/design
```
